# 🩺 Flutter Doctor App

A modern Flutter doctor appointment application built with **Clean Architecture**, **Firebase**, **BLoC**, **Provider**, and **Riverpod**. The app provides a scalable, maintainable, and production-ready foundation for healthcare applications.

## ✨ Features

- 🔐 Firebase Authentication
- 👨‍⚕️ Browse doctors
- 📅 Book doctor appointments
- 👤 User profile management
- ❤️ Favorite doctors
- 🔍 Search functionality
- 📱 Responsive and clean UI
- ☁️ Firebase integration
- 🏗️ Clean Architecture
- ⚡ State management using BLoC, Cubit

## 🛠 Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Clean Architecture
- BLoC
- DIO
- Retrofit

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio or VS Code
- Firebase Project

### Installation

```bash
git clone https://github.com/your-username/flutter-doctor-app.git
cd flutter-doctor-app
flutter pub get
```

Run the application:

```bash
flutter run
```
